Xây dựng Database Schema (EF Core):

Triển khai thành công 9 bảng dữ liệu cốt lõi thông qua cơ chế Migration.

Các cập nhật quan trọng: Bổ sung địa chỉ khách hàng (themDiaChiKhachHang), cập nhật thông tin nhân viên (themDiaChiNV), và chi tiết bảng sản phẩm.

Đảm bảo tính toàn vẹn dữ liệu giữa các bảng: LoaiDongHo, HangSanXuat, SanPham, HoaDon, ChiTietHoaDon...

Thiết kế Giao diện Hệ thống:

Hoàn thiện bộ khung các Form quản lý: frmSanPham, frmNhanVien, frmKhachHang, frmThuongHieu.

Thiết kế giao diện frmTrangChu (Main Dashboard) làm nơi điều hướng chính.

Xây dựng giao diện frmDangNhap bảo mật cho hệ thống.

Quản lý tài nguyên:

Thiết lập thư mục Resources để quản lý hình ảnh hiển thị cho sản phẩm đồng hồ, giúp UI sinh động hơn.
